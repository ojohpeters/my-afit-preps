рҹ”ў PART 1 вҖ” Indices (Laws of Powers)

1. What is an index?

An index (plural: indices), also called a power or exponent, tells you how many times a number multiplies itself.

aвҒҝ = a Г— a Г— a Г— вҖҰ (n times)

In aвҒҝ: the bottom number a is the base, and the small raised number n is the index/power.

  • 2Ві means 2 Г— 2 Г— 2 = 8. Here the base is 2 and the index is 3.
  • 5ВІ means 5 Г— 5 = 25.
  • aвҒҙ means a Г— a Г— a Г— a.
рҹҢҹ Memory: the index just counts how many copies of the base are multiplied together. It is NOT the base times the index вҖ” 2Ві is 8, not 6.

2. The seven laws of indices

These laws let you combine powers without writing them all out. Learn this table by heart вҖ” nearly every indices question uses one of them.

3. Worked examples вҖ” simplifying

Example 1: Simplify xвҒҙ Г— xвҒө.
Same base x, so ADD the powers: xвҒҙ Г— xвҒө = xвҒҙвҒәвҒө = xвҒ№
Answer = xвҒ№.
Example 2: Simplify (2aВІ)Ві.
Cube everything inside: 2Ві Г— (aВІ)Ві = 8 Г— aвҒ¶
Answer = 8aвҒ¶ (remember to cube the 2 as well).
Example 3: Simplify (aвҒ¶ Г— aВІ) Г· aВі.
Top: aвҒ¶ Г— aВІ = aвҒё
Then divide: aвҒё Г· aВі = aвҒёвҒ»Ві = aвҒө
Answer = aвҒө.

4. Worked examples вҖ” evaluating (getting a number)

Example 1: Evaluate 2Ві Г— 2ВІ.
Add powers: 2ВівҒәВІ = 2вҒө = 2Г—2Г—2Г—2Г—2 = 32
Answer = 32.
Example 2: Evaluate 27^(2/3).
Denominator = root, numerator = power: (ВівҲҡ27)ВІ
ВівҲҡ27 = 3, then 3ВІ = 9
Answer = 9.
Example 3: Evaluate 16^(вҲ’1/2).
Negative вҶ’ flip: 16^(вҲ’1/2) = 1 Г· 16^(1/2) = 1 Г· вҲҡ16
вҲҡ16 = 4, so answer = 1/4
Answer = Вј.
Example 4: Evaluate 5вҒ° + 2вҒ»В№.
5вҒ° = 1 and 2вҒ»В№ = 1/2
1 + ВҪ = 3/2 = 1ВҪ
Answer = 1ВҪ.

5. Solving indicial (index) equations

An indicial equation has the unknown in the power, like 2ЛЈ = 32. The trick: make BOTH sides have the same base, then the powers must be equal.

If aЛЈ = aвҒҝ, then x = n
Example 1: Solve 2ЛЈ = 32.
Write 32 as a power of 2: 32 = 2вҒө
So 2ЛЈ = 2вҒө вҶ’ x = 5
Answer = x = 5.
Example 2: Solve 3^(x+1) = 81.
81 = 3вҒҙ, so 3^(x+1) = 3вҒҙ
Powers equal: x + 1 = 4 вҶ’ x = 3
Answer = x = 3.
Example 3: Solve 4ЛЈ = 8.
Use base 2: 4 = 2ВІ and 8 = 2Ві, so (2ВІ)ЛЈ = 2Ві
2^(2x) = 2Ві вҶ’ 2x = 3 вҶ’ x = 3/2
Answer = x = 1ВҪ.

рҹ“Ҳ PART 2 вҖ” Logarithms

1. What is a logarithm?

A logarithm answers the question: "to what power must I raise the base to get this number?" It is simply the inverse (opposite) of an index.

If aЛЈ = b, then loga b = x  (and back again)

So the log and the power are two ways of saying the same fact:

  • Because 2Ві = 8, we can write logвӮӮ 8 = 3.
  • Because 10ВІ = 100, we can write logвӮҒвӮҖ 100 = 2.
  • Because 5ВІ = 25, we can write logвӮ… 25 = 2.
рҹҢҹ Memory: the log gives you back the power. "log base a of b" asks: a to what power equals b? Read it as the missing exponent.

2. Common logarithms (base 10)

When no base is written, it usually means base 10 вҖ” the common logarithm. So "log 1000" means logвӮҒвӮҖ 1000.

log 10 = 1  (because 10В№ = 10)
log 100 = 2  (because 10ВІ = 100)
log 1000 = 3  (because 10Ві = 1000)

Notice: for powers of 10, the common log is just the number of zeros. That is why base 10 was so useful in the old log tables.

3. The laws of logarithms

These mirror the laws of indices (because logs ARE indices in disguise). Multiplying numbers becomes ADDING their logs; dividing becomes SUBTRACTING.

Log of baseloga a = 1aВ№ = a, so the answer is 1

4. Worked examples вҖ” evaluating & using the laws

Example 1: Evaluate logвӮӮ 8.
Ask: 2 to what power gives 8? 2Ві = 8
Answer = 3.
Example 2: Evaluate logвӮҒвӮҖ 1000.
10 to what power gives 1000? 10Ві = 1000
Answer = 3.
Example 3 вҖ” use the laws: Simplify log 6 + log 4 вҲ’ log 3.
Add вҶ’ multiply, subtract вҶ’ divide: log(6 Г— 4 Г· 3)
= log(24 Г· 3) = log 8
Answer = log 8.
Example 4 вҖ” power law: Write 2 log 5 + log 4 as a single log.
2 log 5 = log 5ВІ = log 25
log 25 + log 4 = log(25 Г— 4) = log 100 = 2
Answer = 2.

5. Solving log equations

Example 1: Solve logвӮғ x = 2.
Turn the log back into a power: x = 3ВІ
x = 9
Answer = x = 9.
Example 2: Solve logвӮӮ x = 5.
x = 2вҒө = 32
Answer = x = 32.

6. Antilogarithm вҖ” going backwards

The antilog undoes a log. If log N = x, then the antilog of x is N = 10ЛЈ (for base 10). It simply takes you back from the log to the original number.

If log N = 2, then N = 10ВІ = 100 (antilog of 2 is 100)
рҹҢҹ Memory: log turns a number into its power; antilog turns the power back into the number. They cancel each other out, just like squaring and square-rooting.

вҲҡ PART 3 вҖ” Surds

1. What is a surd?

A surd is a root (usually a square root) that cannot be simplified to an exact whole number or fraction вҖ” its value is an irrational never-ending, non-repeating decimal, so we leave it in root form.

  • вҲҡ2, вҲҡ3, вҲҡ5, вҲҡ7 are surds вҖ” вҲҡ2 = 1.41421вҖҰ goes on forever.
  • вҲҡ4 = 2, вҲҡ9 = 3, вҲҡ25 = 5 are NOT surds вҖ” they give exact whole numbers (perfect squares).
рҹҢҹ Memory: if the number under the root is a perfect square (4, 9, 16, 25вҖҰ), the root is a whole number and it is NOT a surd. Otherwise it stays as a surd.

2. The two basic rules

RuleExample
вҲҡ(a Г— b) = вҲҡa Г— вҲҡbвҲҡ6 = вҲҡ2 Г— вҲҡ3
вҲҡ(a Г· b) = вҲҡa Г· вҲҡbвҲҡ(9/4) = вҲҡ9 Г· вҲҡ4 = 3/2

These let us split a surd into pieces вҖ” which is exactly how we simplify.

3. Simplifying surds

To simplify a surd, split off the largest perfect square hidden inside it, then take its root out.

Example 1: Simplify вҲҡ50.
50 = 25 Г— 2, and 25 is a perfect square
вҲҡ50 = вҲҡ25 Г— вҲҡ2 = 5вҲҡ2
Answer = 5вҲҡ2.
Example 2: Simplify вҲҡ18.
18 = 9 Г— 2 вҶ’ вҲҡ18 = вҲҡ9 Г— вҲҡ2 = 3вҲҡ2
Answer = 3вҲҡ2.
Example 3: Simplify вҲҡ72.
72 = 36 Г— 2 вҶ’ вҲҡ72 = вҲҡ36 Г— вҲҡ2 = 6вҲҡ2
Answer = 6вҲҡ2.

4. Adding and subtracting surds

You can only add or subtract like surds (the same root part) вҖ” just like adding "apples to apples". Simplify first so hidden like surds appear.

Example 1: 3вҲҡ2 + 5вҲҡ2.
Same вҲҡ2, so add the numbers: (3 + 5)вҲҡ2 = 8вҲҡ2
Answer = 8вҲҡ2.
Example 2: вҲҡ50 + вҲҡ18.
Simplify each: вҲҡ50 = 5вҲҡ2, вҲҡ18 = 3вҲҡ2
5вҲҡ2 + 3вҲҡ2 = 8вҲҡ2
Answer = 8вҲҡ2.

Warning: вҲҡ2 + вҲҡ3 does NOT combine вҖ” they are unlike surds, so leave it as вҲҡ2 + вҲҡ3.

5. Multiplying surds

Example 1: вҲҡ3 Г— вҲҡ3.
вҲҡa Г— вҲҡa = a, so вҲҡ3 Г— вҲҡ3 = 3
Answer = 3.
Example 2: вҲҡ2 Г— вҲҡ8.
вҲҡ2 Г— вҲҡ8 = вҲҡ(2 Г— 8) = вҲҡ16 = 4
Answer = 4.
Example 3: 2вҲҡ3 Г— 4вҲҡ5.
Multiply numbers and roots separately: (2 Г— 4)(вҲҡ3 Г— вҲҡ5) = 8вҲҡ15
Answer = 8вҲҡ15.

6. Rationalising the denominator

Rationalising means clearing the surd out of the bottom of a fraction, so the denominator becomes a whole number. It is a standard exam requirement.

(a) Single surd on the bottom

Multiply top and bottom by that same surd.

Example 1: Rationalise 1/вҲҡ2.
Multiply top & bottom by вҲҡ2: (1 Г— вҲҡ2) Г· (вҲҡ2 Г— вҲҡ2)
= вҲҡ2 Г· 2 = вҲҡ2/2
Answer = вҲҡ2/2.
Example 2: Rationalise 6/вҲҡ3.
Multiply by вҲҡ3: (6вҲҡ3) Г· (вҲҡ3 Г— вҲҡ3) = 6вҲҡ3 Г· 3
= 2вҲҡ3
Answer = 2вҲҡ3.

(b) Using the conjugate (a + вҲҡb type)

When the bottom is like 3 вҲ’ вҲҡ2, multiply top and bottom by its conjugate (same terms, opposite sign: 3 + вҲҡ2). This uses (x вҲ’ y)(x + y) = xВІ вҲ’ yВІ to remove the surd.

Example 3: Rationalise 1/(3 вҲ’ вҲҡ2).
Multiply top & bottom by the conjugate (3 + вҲҡ2):
Bottom: (3 вҲ’ вҲҡ2)(3 + вҲҡ2) = 3ВІ вҲ’ (вҲҡ2)ВІ = 9 вҲ’ 2 = 7
Top: 1 Г— (3 + вҲҡ2) = 3 + вҲҡ2
Answer = (3 + вҲҡ2)/7.
рҹҢҹ Memory: the conjugate just flips the middle sign. Multiplying a surd pair by its conjugate always kills the surd, leaving a clean whole number on the bottom.

рҹ§© PART 4 вҖ” Mixed & Exam-Style Problems

1. Linking indices and logs

Example: Given that logвӮӮ 8 = x, find x, then find 2ЛЈ.
logвӮӮ 8 asks: 2 to what power = 8? 2Ві = 8, so x = 3
Then 2ЛЈ = 2Ві = 8 (we simply return to where we started)
Answer: x = 3, and 2ЛЈ = 8. This shows log and index are inverses.

2. Harder indicial equation

Example: Solve 9ЛЈ = 27.
Use base 3: 9 = 3ВІ and 27 = 3Ві
(3ВІ)ЛЈ = 3Ві вҶ’ 3^(2x) = 3Ві вҶ’ 2x = 3
x = 3/2
Answer = x = 1ВҪ.

3. Combining log laws to a number

Example: Evaluate log 2 + log 50 (base 10).
Add вҶ’ multiply inside: log(2 Г— 50) = log 100
log 100 = 2 (because 10ВІ = 100)
Answer = 2.

4. Surd expression with rationalising

Example: Simplify (вҲҡ5)ВІ + вҲҡ16 вҲ’ 1/вҲҡ2 rationalised.
(вҲҡ5)ВІ = 5 and вҲҡ16 = 4, so far 5 + 4 = 9
1/вҲҡ2 rationalised = вҲҡ2/2, so the value is 9 вҲ’ вҲҡ2/2
Answer = 9 вҲ’ вҲҡ2/2 (an exact surd form is the expected answer).
Example вҖ” pure evaluation: Simplify вҲҡ98 вҲ’ вҲҡ8.
вҲҡ98 = вҲҡ(49 Г— 2) = 7вҲҡ2, and вҲҡ8 = вҲҡ(4 Г— 2) = 2вҲҡ2
7вҲҡ2 вҲ’ 2вҲҡ2 = 5вҲҡ2
Answer = 5вҲҡ2.
