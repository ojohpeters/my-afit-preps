📋 PART 1 — Data & the Three Averages

1. What is statistics? Types of data

Statistics is the branch of maths that deals with collecting, organising, presenting and interpreting data (information, usually numbers).

Ways we split up data
  • Qualitative (categorical) data — describes a quality, not a number: colour of a car, favourite fruit, gender, blood group.
  • Quantitative (numerical) data — a measured or counted number: height, mass, age, number of goals.
  • Discrete data — counted, so it takes whole-number values only: number of children, number of cars (you cannot have 2.5 children).
  • Continuous data — measured, so it can take any value in a range: height (1.72 m), mass, temperature, time.
🌟 Memory: you COUNT discrete data (whole numbers) but you MEASURE continuous data (decimals allowed).

2. The three averages (measures of central tendency)

An average is a single value that represents the "middle" or "typical" value of a data set. There are three:

  • Mean — the arithmetic average; add everything up and share it out.
  • Median — the middle value when the data is put in order.
  • Mode — the value that appears most often.
🌟 Remember the M-words: Mean = add and share; Median = middle; Mode = most.

3. The MEAN — add up, divide by how many

The mean is the total of all the values divided by the number of values.

Mean (x̄) = Σx ÷ n

Σx means "the sum of all the x values" and n is how many values there are.

Worked example: Find the mean of 4, 8, 6, 10, 2.
Σx = 4 + 8 + 6 + 10 + 2 = 30
n = 5 values
Mean = 30 ÷ 5 = 6
The mean is 6.
Working backwards: The mean of 5 numbers is 12. Find their total.
Total = mean × n = 12 × 5 = 60
The numbers add up to 60. (Rearranging Σx = mean × n is a very common exam trick.)

4. The MEDIAN — the middle value

Put the numbers in order first (smallest to largest). The median is the one in the middle.

Odd number of values

If there is an odd count, there is one single middle value. Its position is (n + 1) ÷ 2.

Odd example: Find the median of 7, 2, 9, 5, 3.
Order: 2, 3, 5, 7, 9
n = 5, middle position = (5 + 1) ÷ 2 = 3rd value
The 3rd value is 5, so the median is 5.
Even number of values

If there is an even count, there are two middle values — the median is their mean (add them and divide by 2).

Even example: Find the median of 2, 4, 6, 8, 10, 12.
Already in order; n = 6, so the two middle values are the 3rd and 4th: 6 and 8
Median = (6 + 8) ÷ 2 = 14 ÷ 2 = 7
The median is 7.
🌟 Always ORDER the data before finding the median — forgetting this is the #1 mistake.

5. The MODE — the most common value

The mode is the value that appears most frequently. A set can have one mode, more than one mode, or no mode at all.

Worked example: Find the mode of 2, 3, 3, 5, 7, 3, 8.
Count each: 2 appears once, 3 appears three times, 5 once, 7 once, 8 once
3 appears the most (three times), so the mode is 3.
  • Two values tie for most common → the data is bimodal (two modes).
  • Every value appears the same number of times → there is no mode.
🌟 The mode is the only average that works for qualitative data (e.g. the most popular colour).

6. Which average to use?

  • The mean uses every value, but it is easily pulled by an extreme value (an outlier). One huge salary can drag the mean up.
  • The median ignores how big the extremes are, so it is not affected by outliers — good for skewed data like incomes.
  • The mode is best for "most popular" questions and for categories.

🗂️ PART 2 — Frequency Tables & Grouped Data

1. Frequency tables (ungrouped)

When a value repeats many times we record how often with a frequency (f) — the number of times it occurs. This saves listing every value.

Mean from a frequency table
Mean = Σfx ÷ Σf

Multiply each value x by its frequency f to get fx, add up the fx column (Σfx), then divide by the total frequency (Σf = n).

Worked example: The table shows marks scored by 20 pupils.
Mark (x)12345Total
Frequency (f)2456320
fx2815241564
Σf = 2 + 4 + 5 + 6 + 3 = 20
Σfx = 2 + 8 + 15 + 24 + 15 = 64
Mean = Σfx ÷ Σf = 64 ÷ 20 = 3.2
The mean mark is 3.2.

2. Mode and median from a frequency table

Using the same table above:

  • Mode = the value with the highest frequency. The biggest f is 6, which belongs to the mark 4, so the mode is 4.
  • Median = the middle value. With n = 20 (even), the median is between the 10th and 11th values. Building up the frequencies (cumulative): up to mark 2 we have 6 pupils; up to mark 3 we have 11 pupils. So both the 10th and 11th pupils scored 3 → the median is 3.
🌟 Do not confuse the mode (the x-value with the biggest frequency) with the frequency itself. The mode is 4, NOT 6.

3. Grouped data — using classes

When data is spread over a wide range (like exam marks out of 100) we group it into classes (intervals) such as 0–10, 10–20, and so on. We lose the exact values but the data becomes manageable.

The class midpoint (mid-value)

Because we no longer know the exact values in a class, we assume they all sit at the midpoint of the class.

Class midpoint = (lower limit + upper limit) ÷ 2
Example: the midpoint of the class 20–30 is
(20 + 30) ÷ 2 = 50 ÷ 2 = 25

4. Estimated mean of a grouped distribution

Use the midpoint of each class as the x-value, then use the same formula Mean = Σfx ÷ Σf. Because midpoints are only estimates of the real values, the answer is called the estimated mean.

Worked example: The table shows the marks of 20 students.

The estimated mean mark is 24.

5. Modal class and median class

With grouped data we cannot give a single mode or median value, so we name the class instead.

  • Modal class = the class with the highest frequency. In the table above the biggest frequency is 8, so the modal class is 20–30.
  • Median class = the class containing the middle value. With n = 20 the median lies at the 10th/11th value. Cumulative frequency reaches 7 by the end of 10–20 and reaches 15 by the end of 20–30, so the 10th and 11th values fall in 20–30. That is the median class.
🌟 Modal class and median class are found the same way as before — just report the interval, not a single number.

📏 PART 3 — Measures of Spread (Dispersion)

1. Why we need spread as well as average

Two sets can have the same mean but be very different. {2, 6, 10} and {5, 6, 7} both have mean 6, but the first is much more spread out. Measures of spread (dispersion) tell us how scattered the data is around the average.

The main ones are the range, the mean deviation, the variance and the standard deviation.

2. The RANGE — the simplest spread

Range = Largest value − Smallest value
Worked example: Find the range of 12, 5, 20, 8, 15.
Largest = 20, Smallest = 5
Range = 20 − 5 = 15
The range is 15.

The range is quick but only uses two values, so a single extreme value can make it misleading.

3. The MEAN DEVIATION

The mean deviation is the average distance of each value from the mean. We ignore the sign (use the absolute value |x − x̄|) because we only care about the distance.

Mean deviation = Σ|x − x̄| ÷ n
Worked example: Find the mean deviation of 2, 4, 6, 8, 10.
Mean x̄ = (2 + 4 + 6 + 8 + 10) ÷ 5 = 30 ÷ 5 = 6
Distances |x − 6|: 4, 2, 0, 2, 4
Σ|x − x̄| = 4 + 2 + 0 + 2 + 4 = 12
Mean deviation = 12 ÷ 5 = 2.4
The mean deviation is 2.4.

4. VARIANCE and STANDARD DEVIATION

The standard deviation is the most important measure of spread. Instead of the absolute value we square the deviations (which also removes minus signs), average them to get the variance, then square-root back to the original units.

Variance (σ²) = Σ(x − x̄)² ÷ n
Standard deviation (σ) = √[ Σ(x − x̄)² ÷ n ]

So the standard deviation is simply the square root of the variance. A small σ means the data is bunched close to the mean; a large σ means it is widely spread.

5. Fully worked standard-deviation example

Find the standard deviation of 2, 4, 6, 8, 10.
Step 1 — mean: x̄ = (2 + 4 + 6 + 8 + 10) ÷ 5 = 30 ÷ 5 = 6

📈 PART 4 — Presenting Data with Charts

1. Bar chart

A bar chart shows data with separate rectangular bars of equal width. The height (or length) of each bar shows the frequency. It is used for discrete or categorical data (e.g. favourite subjects).

  • The bars have gaps between them (this is what separates a bar chart from a histogram).
  • Bars can be vertical or horizontal.

2. Pie chart

A pie chart is a circle split into sectors. Each sector's angle is proportional to the frequency it represents. The whole circle is 360°.

Sector angle = (frequency ÷ total frequency) × 360°
Worked example: 60 students chose a favourite fruit — Apple 20, Banana 15, Orange 10, Mango 15. Find each sector angle.
Apple = (20 ÷ 60) × 360° = 120°
Banana = (15 ÷ 60) × 360° = 90°
Orange = (10 ÷ 60) × 360° = 60°
Mango = (15 ÷ 60) × 360° = 90°
Check: 120° + 90° + 60° + 90° = 360° ✓
Always check your angles add up to 360°.
🌟 To turn a sector angle back into a percentage: percentage = (angle ÷ 360°) × 100.

3. Histogram

A histogram is used for grouped continuous data. It looks like a bar chart but the bars touch with no gaps, because the classes run continuously.

  • The horizontal axis shows the class boundaries.
  • For equal class widths, the bar height = frequency. (For unequal widths, the area of the bar represents the frequency and the height is the frequency density = frequency ÷ class width.)
  • The tallest bar corresponds to the modal class.

4. Frequency polygon

A frequency polygon is a line graph made by plotting the frequency against the class midpoint and joining the points with straight lines.

  • It is often drawn over a histogram by joining the midpoints of the tops of the bars.
  • It is useful for comparing two or more distributions on the same axes.

5. Cumulative frequency curve (ogive)

The cumulative frequency is a running total — you add each frequency to the ones before it. Plotting the cumulative frequency against the upper class boundary and joining the points with a smooth curve gives the ogive (cumulative frequency curve), an S-shaped graph.

Reading values off the ogive

Let n be the total frequency (the top of the curve). Read across from these positions on the cumulative-frequency axis, down to the curve, then down to the value axis:

  • Median — read across at the n⁄2 position (the middle).
  • Lower quartile (Q₁) — read across at the n⁄4 position.
  • Upper quartile (Q₃) — read across at the 3n⁄4 position.
  • Interquartile range = Q₃ − Q₁ (a measure of spread that ignores extremes).
Example positions: if n = 40 students, then
Median at 40 ÷ 2 = 20th value
Q₁ at 40 ÷ 4 = 10th value
Q₃ at (3 × 40) ÷ 4 = 30th value
🌟 The ogive always rises (never falls) because you keep adding on — it can only stay level or go up.
