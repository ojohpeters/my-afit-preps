📈 Direct & Inverse Variation

1. What does “variation” mean?

Variation describes how two quantities are connected — when one changes, the other changes in a fixed, predictable way. Maths uses the symbol ∝, which reads “varies as” or “is proportional to”.

🌟 Real life: the more petrol you buy, the more you pay (they rise together). The more taps you open on a tank, the less time it takes to fill (one rises, the other falls). These fixed patterns are exactly what variation captures.
The golden rule

Every ∝ statement becomes an equation by replacing ∝ with “= k ×”, where k is a fixed number called the constant of proportionality (or constant of variation). Your whole job in a variation question is usually: find k, then use it.

y ∝ x  means  y = k·x

2. DIRECT variation  ( y ∝ x → y = kx )

In direct variation the two quantities rise and fall together. If x doubles, y doubles; if x is cut to a third, y is cut to a third. The ratio y ÷ x always equals k.

The 3-step method (memorise this)
  • Step 1 – Find k: write y = kx, put in the pair of values you are given, and solve for k.
  • Step 2 – Write the equation: put that k back to get the full rule (e.g. y = 4x).
  • Step 3 – Use it: substitute the new value to get the answer.
Worked Example 1: y varies directly as x. When x = 5, y = 20. Find y when x = 8.
Step 1 – find k: y = kx → 20 = k×5 → k = 20÷5 = 4
Step 2 – equation: y = 4x
Step 3 – use it: x = 8 → y = 4×8 = 32
Answer: y = 32
Worked Example 2 (word problem): The cost C of oranges varies directly as the number n bought. 6 oranges cost ₦150. Find the cost of 10 oranges.
Step 1: C = kn → 150 = k×6 → k = 150÷6 = 25 (₦ per orange)
Step 2: C = 25n
Step 3: n = 10 → C = 25×10 = ₦250
Answer: ₦250

3. INVERSE variation  ( y ∝ 1/x → y = k/x )

In inverse variation the quantities move in opposite directions. As x increases, y decreases; if x doubles, y is halved. The product x × y stays constant and equals k.

y ∝ 1/x  means  y = k/x   (so k = xy)
💡 Trick: for inverse variation, k is simply the two given numbers multiplied together (k = xy). For direct variation, k is one divided by the other (k = y ÷ x).
Worked Example 1: y varies inversely as x. When x = 4, y = 3. Find y when x = 6.
Step 1: y = k/x → 3 = k/4 → k = 3×4 = 12
Step 2: y = 12/x
Step 3: x = 6 → y = 12÷6 = 2
Answer: y = 2
Worked Example 2 (word problem): The time t to finish a job varies inversely as the number of men m. 8 men take 10 days. How long will 5 men take?
Step 1: t = k/m → 10 = k/8 → k = 10×8 = 80
Step 2: t = 80/m
Step 3: m = 5 → t = 80÷5 = 16 days
Answer: 16 days  (fewer men → more time, as expected)

4. Variation with POWERS  ( x², √x, 1/x² )

Sometimes y varies as a power of x. The method is exactly the same — the power just travels with the x.

StatementEquationMeaning
y varies as the square of xy = kx²uses x²
y varies as the square root of xy = k√xuses √x
y varies inversely as the square of xy = k/x²x² on the bottom
Example (square, y ∝ x²): y varies as x². When x = 3, y = 18. Find y when x = 5.
y = kx² → 18 = k×3² = k×9 → k = 18÷9 = 2
y = 2x² → x = 5: y = 2×5² = 2×25 = 50
Answer: y = 50
Example (root, y ∝ √x): y varies as √x. When x = 16, y = 12. Find y when x = 25.
y = k√x → 12 = k×√16 = k×4 → k = 12÷4 = 3
y = 3√x → x = 25: y = 3×√25 = 3×5 = 15
Answer: y = 15
Example (inverse square, y ∝ 1/x²): y varies inversely as x². When x = 3, y = 4. Find y when x = 6.
y = k/x² → 4 = k/3² = k/9 → k = 4×9 = 36
y = 36/x² → x = 6: y = 36÷6² = 36÷36 = 1
Answer: y = 1

🔗 Joint & Combined Variation

1. What is joint variation?

Joint variation is when one quantity depends on two or more other quantities at the same time, all multiplied together with a single constant k.

y varies jointly as x and z  →  y = kxz
🌟 Real life: the wages you earn depend jointly on the number of days worked and the rate per day. Both go up → wages go up. That is joint variation.
Same 3-step method

Find k from the first complete set of values, write the equation, then substitute the new values. The only difference is that you plug in two (or more) numbers instead of one.

2. Combined variation (direct AND inverse together)

Very often a quantity varies directly as one thing and inversely as another. Directly-as goes on top, inversely-as goes underneath.

y varies directly as x and inversely as z  →  y = kx/z

⚠️ Read the words carefully: “jointly as x and z” = kxz (both on top), but “directly as x, inversely as z” = kx/z (z on the bottom).

3. Fully worked examples

Example 1 (pure joint): y varies jointly as x and z. When x = 2, z = 3, y = 24. Find y when x = 4, z = 5.
Step 1: y = kxz → 24 = k×2×3 = 6k → k = 24÷6 = 4
Step 2: y = 4xz
Step 3: x = 4, z = 5 → y = 4×4×5 = 80
Answer: y = 80
Example 2 (joint with a square): y varies jointly as x and the square of z. When x = 3, z = 2, y = 36. Find y when x = 5, z = 3.
Step 1: y = kxz² → 36 = k×3×2² = k×3×4 = 12k → k = 36÷12 = 3
Step 2: y = 3xz²
Step 3: x = 5, z = 3 → y = 3×5×3² = 3×5×9 = 135
Answer: y = 135
Example 3 (combined, direct & inverse): y varies directly as x and inversely as z. When x = 8, z = 2, y = 12. Find y when x = 10, z = 5.
Step 1: y = kx/z → 12 = k×8/2 = 4k → k = 12÷4 = 3
Step 2: y = 3x/z
Step 3: x = 10, z = 5 → y = 3×10/5 = 30÷5 = 6
Answer: y = 6
Example 4 (percentage / factor change — no numbers for k needed): y varies jointly as x and z. If x is increased by 20% and z is decreased by 10%, find the percentage change in y.
y = kxz. New x = 1.20x, new z = 0.90z
New y = k(1.20x)(0.90z) = (1.20×0.90)·kxz = 1.08·kxz
1.08 means the new y is 108% of the old → an 8% increase
Answer: y increases by 8%
Example 5 (factor change with a power): y varies directly as x and inversely as the square of z. If x is doubled and z is halved, what happens to y?
y = kx/z². New x = 2x, new z = ½z → z² = (½z)² = ¼z²
New y = k(2x)/(¼z²) = 2 ÷ ¼ × (kx/z²) = 8·(kx/z²) = 8y
Answer: y becomes 8 times as large

🧩 Partial Variation — the big one

1. What is partial variation?

Partial variation is when a quantity is made of two parts joined by a plus sign: one part is constant (it never changes) and the other part varies (it changes with x). The most common form is:

y = a + bx   (a = the fixed part,   bx = the varying part)

Here there are two unknown constants, a and b. Notice a is not multiplied by x — it stands alone.

🌟 Real life is FULL of partial variation:
  • Taxi fare = a fixed drop charge + a rate per kilometre.
  • Electricity bill = a fixed standing charge + a charge per unit used.
  • Cost of production = fixed overhead (rent, machines) + a cost per item made.
  • Printing = a fixed set-up fee + a cost per copy.
Whenever you see “a fixed charge PLUS a rate”, it is partial variation.

2. Other partial forms

The two parts do not have to be “constant + straight line”. Two parts of any kind can be combined:

WordsEquation
Partly constant, partly varies as xy = a + bx
Partly constant, partly varies inversely as xy = a + b/x
Partly varies as x, partly as the square of xy = ax + bx²

Whatever the form, there are always two unknown constants to find.

3. THE METHOD (why you always get TWO equations)

Because there are two unknowns (a and b), you need two equations to pin them down — and to get two equations you must be given two sets of data. You then solve them simultaneously.

Step 1: Write the general form (e.g. y = a + bx).
Step 2: Put in the FIRST set of data → equation ①.
Step 3: Put in the SECOND set of data → equation ②.
Step 4: Solve ① and ② SIMULTANEOUSLY (subtract to remove a, find b, then find a).
Step 5: Write the complete equation.
Step 6: Use it to answer the question.

💡 Fast tip: when the fixed part a appears in both equations, subtracting equation ① from ② cancels a immediately and hands you b.

4. Fully worked examples (easy → hard)

Example 1 (basic linear partial): y is partly constant and partly varies as x. When x = 2, y = 7; when x = 4, y = 11. Find the equation and hence y when x = 6.
General form: y = a + bx
① x = 2, y = 7: 7 = a + 2b
② x = 4, y = 11: 11 = a + 4b
② − ①: (a + 4b) − (a + 2b) = 11 − 7 → 2b = 4 → b = 2
Put b = 2 into ①: 7 = a + 2(2) = a + 4 → a = 3
Complete equation: y = 3 + 2x
x = 6: y = 3 + 2(6) = 3 + 12 = 15
Answer: y = 3 + 2x, and y = 15 when x = 6
Example 2 (real-life, bigger numbers): The cost C of running a school is partly constant and partly varies as the number of students n. When n = 100, C = ₦800,000; when n = 150, C = ₦1,050,000. Find C when n = 200.
General form: C = a + bn
① 800000 = a + 100b
② 1050000 = a + 150b
② − ①: 250000 = 50b → b = 250000÷50 = 5000
Put b = 5000 into ①: 800000 = a + 100(5000) = a + 500000 → a = 300000
Complete equation: C = 300000 + 5000n
n = 200: C = 300000 + 5000(200) = 300000 + 1,000,000 = ₦1,300,000
Answer: fixed cost ₦300,000, ₦5,000 per student; C = ₦1,300,000 for 200 students
Example 3 (partial with a square, y = ax + bx²): y partly varies as x and partly as the square of x. When x = 2, y = 14; when x = 3, y = 27. Find y when x = 5.
General form: y = ax + bx²
① x = 2: 14 = 2a + 4b → divide by 2: 7 = a + 2b
② x = 3: 27 = 3a + 9b
From ①: a = 7 − 2b. Substitute into ②:
27 = 3(7 − 2b) + 9b = 21 − 6b + 9b = 21 + 3b → 3b = 6 → b = 2
a = 7 − 2(2) = 3
Complete equation: y = 3x + 2x²
x = 5: y = 3(5) + 2(5²) = 15 + 2(25) = 15 + 50 = 65
Answer: y = 3x + 2x², and y = 65 when x = 5
Example 4 (harder — solve, then work BACKWARDS): A printer charges partly a fixed fee and partly a rate per page. Printing 200 pages costs ₦3,500; printing 500 pages costs ₦6,500. (a) Find the fixed fee and the cost per page. (b) How many pages can be printed for ₦8,000?
General form: C = a + bn
① 3500 = a + 200b
② 6500 = a + 500b
② − ①: 3000 = 300b → b = 3000÷300 = 10
① : 3500 = a + 200(10) = a + 2000 → a = 1500
(a) Complete equation: C = 1500 + 10n → fixed fee ₦1,500, ₦10 per page
(b) Set C = 8000: 8000 = 1500 + 10n → 10n = 6500 → n = 650
Answer: fixed fee ₦1,500 and ₦10/page; 650 pages for ₦8,000

Direct, inverse, joint and combined each need one constant (k) and one data set. Partial variation is special: two constants, two data sets, solved simultaneously.
