📉 Differentiation — Finding the Rate of Change

1. What is differentiation?

Differentiation finds how fast one quantity changes compared to another — the rate of change. On a graph, it gives the gradient (slope) of the curve at a point.

🌟 Real life: when you drive, your speedometer shows how fast your distance is changing each second — that is differentiation (speed = rate of change of distance). The steepness of a hill at the exact spot you stand is also a derivative.
Notation

If y depends on x (written y = f(x)), the derivative is written dy/dx or f′(x). It means “the gradient of y with respect to x.”

2. The POWER RULE (your main tool)

If y = a·xⁿ  then  dy/dx = a·n·xⁿ⁻¹

In words: multiply by the power, then reduce the power by 1.

Function ydy/dxWhat happened

⚠️ Two facts students forget: the derivative of a constant is 0, and the derivative of x is 1.

3. Differentiating term by term (sum/difference rule)

Differentiate each term separately, then join with the same + and − signs.

Worked example: y = x³ + 2x² − 5x + 7
d/dx(x³) = 3x²
d/dx(2x²) = 4x
d/dx(−5x) = −5
d/dx(7) = 0
dy/dx = 3x² + 4x − 5

4. The second derivative

Differentiate again to get the second derivative, written d²y/dx² or f″(x). It measures how the gradient itself is changing (used for max/min and acceleration).

Example: y = x⁴
dy/dx = 4x³  →  d²y/dx² = 12x²

5. The CHAIN RULE (function inside a function)

If y = [f(x)]ⁿ  then  dy/dx = n·[f(x)]ⁿ⁻¹ × f′(x)

In words: differentiate the outside (power rule), then multiply by the derivative of the inside (the bracket).

Worked example: y = (2x + 3)⁵
Outside: 5(2x+3)⁴  |  Inside derivative: d/dx(2x+3) = 2
dy/dx = 5(2x+3)⁴ × 2 = 10(2x+3)⁴

6. PRODUCT RULE & QUOTIENT RULE

Product rule (two things multiplied)
If y = u·v  then  dy/dx = u·(dv/dx) + v·(du/dx)
Example: y = x²(2x + 1). Let u = x², v = 2x+1.
du/dx = 2x, dv/dx = 2
dy/dx = x²(2) + (2x+1)(2x) = 2x² + 4x² + 2x = 6x² + 2x
Quotient rule (one thing divided by another)
If y = u/v  then  dy/dx = [v·(du/dx) − u·(dv/dx)] / v²

Memory: “bottom × d(top) − top × d(bottom), all over bottom squared.”

📈 Applications of Differentiation

1. Gradient of a curve at a point

To find the slope at a particular point, find dy/dx, then substitute the x-value.

Example: Find the gradient of y = x² − 3x at x = 2.
dy/dx = 2x − 3
At x = 2: gradient = 2(2) − 3 = 1

2. Equation of tangent and normal

  • Tangent — a line just touching the curve. Its slope m = dy/dx at the point.
  • Normal — the line perpendicular to the tangent. Its slope = −1/m.
  • Use the straight-line formula: y − y₁ = m(x − x₁).
Example: Tangent to y = x² at the point (1, 1).
dy/dx = 2x → at x=1, m = 2
y − 1 = 2(x − 1) → y = 2x − 1
Normal slope = −1/2, so normal: y − 1 = −½(x − 1).

3. Maximum & minimum (turning points)

At a turning point the curve is momentarily flat, so the gradient is zero.

Step 1: set dy/dx = 0 and solve for x (these are the turning points)
Step 2: use d²y/dx² → if positive = MINIMUM, if negative = MAXIMUM
💡 Memory: a minimum is a valley (smiley curve, second derivative +). A maximum is a hilltop (sad curve, second derivative −).
Worked example: Find and classify the turning points of y = x³ − 3x.
dy/dx = 3x² − 3 = 0 → x² = 1 → x = 1 or x = −1
d²y/dx² = 6x
At x = 1: 6(1) = +6 → MINIMUM
At x = −1: 6(−1) = −6 → MAXIMUM

4. Increasing & decreasing functions

  • If dy/dx > 0 → the function is increasing (going up).
  • If dy/dx < 0 → the function is decreasing (going down).

5. Motion: velocity & acceleration

If displacement s is a function of time t:

velocity v = ds/dt   |   acceleration a = dv/dt = d²s/dt²
Example: s = 3t² + 2t. Find v and a.
v = ds/dt = 6t + 2
a = dv/dt = 6 (constant)

6. Rates of change (connected rates)

Differentiation links how quantities change together. For example, if the radius of a balloon grows, its volume grows too — the chain rule connects the two rates: dV/dt = (dV/dr) × (dr/dt).

∫ Integration — The Reverse of Differentiation

1. What is integration?

Integration is the opposite of differentiation (anti-differentiation). It also finds the area under a curve.

🌟 If differentiation breaks distance into speed, integration builds speed back into distance. They undo each other — like × and ÷, or + and −.

2. The POWER RULE for integration

∫ xⁿ dx = xⁿ⁺¹ / (n + 1) + C  (as long as n ≠ −1)

In words: add 1 to the power, then divide by the new power. (It is the power rule done backwards.) Always add the constant + C for an indefinite integral.

∫ … dxResult
∫ x² dxx³/3 + C
∫ 3x² dxx³ + C
∫ (2x + 3) dxx² + 3x + C
∫ x⁻¹ dx (= 1/x)ln|x| + C  (special case)

⚠️ Never forget the + C on an indefinite integral — it is a common place to lose marks.

3. Standard integrals to memorise

∫ … dxResult
∫ cos x dxsin x + C
∫ sin x dx−cos x + C
∫ sec²x dxtan x + C
∫ eˣ dxeˣ + C
∫ (1/x) dxln|x| + C

Notice the sign flip: ∫sin = −cos (differentiation of cos gives −sin, so reverse it).

4. DEFINITE integrals (with limits) — gives a number

∫ₐᵇ f(x) dx = F(b) − F(a)

Integrate as usual, then put in the top limit, the bottom limit, and subtract. No + C needed (it cancels out).

Worked example: Evaluate ∫₀² x² dx.
∫ x² dx = x³/3
= [x³/3]₀² = (2³/3) − (0³/3) = 8/3 − 0 = 8/3
Example 2: ∫₁³ 2x dx
∫2x dx = x² → [x²]₁³ = 3² − 1² = 9 − 1 = 8

5. Area under a curve

The area between a curve y = f(x), the x-axis, and the lines x = a and x = b is exactly the definite integral:

Area = ∫ₐᵇ y dx
Example: Area under y = x² from x = 0 to x = 2 is ∫₀² x² dx = 8/3 square units (from the example above).

If part of the curve is below the x-axis, that part gives a negative value — handle it separately and take the size (modulus).

6. Finding the constant C (if given a point)

Example: dy/dx = 2x and the curve passes through (1, 5). Find y.
y = ∫2x dx = x² + C
Put in (1, 5): 5 = 1² + C → C = 4
So y = x² + 4

🎓 Further Maths Extras

1. Integration by substitution (reverse chain rule)

When a function is “inside” another, let u = the inside, replace dx using du, integrate in u, then put x back.

Example: ∫ 2x(x² + 1)³ dx. Let u = x² + 1 → du = 2x dx.
= ∫ u³ du = u⁴/4 + C = (x² + 1)⁴ / 4 + C

2. Integration by parts (reverse product rule)

∫ u dv = uv − ∫ v du

Use it for products like x·eˣ or x·sin x. Pick u using LIATE (Log, Inverse, Algebra, Trig, Exponential — choose u as the one earliest in that list).

Example: ∫ x·eˣ dx. Let u = x (du = dx), dv = eˣ dx (v = eˣ).
= x·eˣ − ∫ eˣ dx = x·eˣ − eˣ + C = eˣ(x − 1) + C

3. Implicit differentiation

When y is not isolated (e.g. x² + y² = 25), differentiate both sides with respect to x, treating y as a function of x (so d/dx of y² is 2y·dy/dx), then make dy/dx the subject.

Example: x² + y² = 25
2x + 2y(dy/dx) = 0 → dy/dx = −x/y

4. Parametric differentiation

When x and y are each given in terms of a parameter t:

dy/dx = (dy/dt) ÷ (dx/dt)
Example: x = t², y = t³.
dx/dt = 2t, dy/dt = 3t²
dy/dx = 3t² ÷ 2t = (3/2)t

5. Volume of revolution

Spin a curve around the x-axis and you get a solid; its volume is:

V = π ∫ₐᵇ y² dx

6. Simple differential equations (separable)

Separate the variables (all y on one side, all x on the other), then integrate both sides.

Example: dy/dx = xy
(1/y) dy = x dx → ∫(1/y)dy = ∫x dx
ln|y| = x²/2 + C
