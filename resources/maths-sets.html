🔢 PART 1 — Sets, Elements & Notation

1. What is a set?

A set is a well-defined collection of distinct (different) objects. The objects are called the elements or members of the set.

Well-defined means there is a clear rule so that we can always tell whether something belongs to the set or not — no arguing. For example, the set of even numbers is well-defined, but the set of tall students is NOT (people disagree on what counts as tall).

We usually name a set with a CAPITAL letter and list its members inside curly braces { }.

A = {2, 4, 6, 8}  ·  B = {a, e, i, o, u}
Two rules to remember
  • Order does not matter. {1, 2, 3} is the same set as {3, 1, 2}.
  • No repeats. Each element is listed once. {1, 1, 2} is just written {1, 2}.
🌟 Idea: think of a set as a labelled box. What matters is WHICH things are inside — not the order you put them in, and you never keep two copies of the same thing.

2. Elements and membership (∈ and ∉)

To say an object is inside a set we use the symbol ∈ (is an element of). To say it is NOT inside we use ∉ (is not an element of).

x ∈ A  means  x is an element of A  ·  x ∉ A  means  x is NOT in A
Worked example: Let A = {2, 4, 6, 8}.
4 ∈ A  (4 is in the set — true)
5 ∉ A  (5 is not listed — true)
So 4 belongs to A, but 5 does not.

3. Two ways to write a set

There are two standard methods of describing a set.

(a) Roster (tabular) method — list every member

You simply write out all the elements inside braces.

C = {1, 3, 5, 7, 9}

(b) Set-builder notation — give a rule

You describe the members by a rule instead of listing them. Read the colon ( : or | ) as such that.

C = {x : x is an odd number less than 10}

This reads: C is the set of all x such that x is an odd number less than 10 — which is exactly {1, 3, 5, 7, 9}. Both methods describe the same set.

🌟 Roster = show me the actual list. Set-builder = tell me the rule and I will build the list myself.

Two things students confuse
  • The empty set ∅ has 0 elements. But {0} is NOT empty — it contains one element, the number 0. And {∅} is not empty either — it contains one element (the empty set itself).
  • The universal set (symbol ξ, sometimes written U) is the biggest box for a problem; every other set in that problem sits inside it.

5. Subsets (⊂) and the power set

Set A is a subset of set B (written A ⊂ B) if every element of A is also in B.

Worked example: Let A = {1, 2} and B = {1, 2, 3, 4}.
Is 1 in B? yes. Is 2 in B? yes.
Every element of A is inside B, so A ⊂ B.
Handy subset facts
  • The empty set ∅ is a subset of every set.
  • Every set is a subset of itself.
  • A set with n elements has 2ⁿ subsets in total.

The power set of A is the set of all possible subsets of A.

Worked example: If A = {a, b}, then n(A) = 2, so the number of subsets is 2² = 4:
Power set = { ∅, {a}, {b}, {a, b} }
Count them: 4 subsets, as predicted by 2ⁿ.

6. Cardinality — n(A)

The cardinality of a set is simply how many elements it has. We write it as n(A).

Worked example:
If A = {5, 10, 15, 20}, then n(A) = 4
If B = {a, e, i, o, u}, then n(B) = 5
For the empty set, n(∅) = 0
(Remember: no repeats are counted twice.)

⚙️ PART 2 — Union, Intersection, Complement & Difference

Throughout this part we use ξ = {1, 2, 3, 4, 5, 6, 7, 8}, A = {1, 2, 3, 4} and B = {3, 4, 5, 6}.

1. Union (∪) — put them together

The union A ∪ B is the set of all elements that are in A, or in B, or in both. Think of the keyword OR.

A ∪ B = {x : x ∈ A OR x ∈ B}
Worked example: A = {1, 2, 3, 4}, B = {3, 4, 5, 6}.
Collect everything from both, shared items listed once:
A ∪ B = {1, 2, 3, 4, 5, 6}
Notice 3 and 4 appear once, not twice.

2. Intersection (∩) — the overlap only

The intersection A ∩ B is the set of elements found in both A and B. Think of the keyword AND.

A ∩ B = {x : x ∈ A AND x ∈ B}
Worked example: A = {1, 2, 3, 4}, B = {3, 4, 5, 6}.
Which numbers are in BOTH lists? 3 and 4.
A ∩ B = {3, 4}

3. Complement (A′) — everything outside

The complement A′ (read A prime, or A complement) is everything in the universal set ξ that is not in A.

A′ = {x : x ∈ ξ AND x ∉ A}
Worked example: ξ = {1, 2, 3, 4, 5, 6, 7, 8}, A = {1, 2, 3, 4}.
Remove A from ξ. What is left?
A′ = {5, 6, 7, 8}

4. Difference (A − B) — take one away from the other

The difference A − B (also written A \ B) is the set of elements in A that are not in B.

A − B = {x : x ∈ A AND x ∉ B}
Worked example: A = {1, 2, 3, 4}, B = {3, 4, 5, 6}.
Start with A = {1, 2, 3, 4}, then throw out anything also in B (3 and 4):
A − B = {1, 2}
Watch out — B − A is different: B − A = {5, 6}. The order matters!

5. Disjoint sets — no overlap

Two sets are disjoint if they have no elements in common, i.e. their intersection is empty.

A and B are disjoint  ⇔  A ∩ B = ∅
Worked example: P = {1, 2, 3} and Q = {7, 8, 9}.
Common elements? none. P ∩ Q = ∅
So P and Q are disjoint sets.

⭕ PART 3 — Venn Diagrams & the Two-Set Formula

1. What is a Venn diagram?

A Venn diagram is a picture that shows sets as overlapping circles inside a rectangle. The rectangle is the universal set ξ; each circle is one set.

  • The overlap of two circles shows A ∩ B (elements in both).
  • The part of A outside the overlap shows A only (A but not B).
  • The region outside all circles (but inside the rectangle) shows elements in neither set.
🌟 Picture two hula-hoops that overlap on the floor. The shared middle patch is the intersection; each hoop-only crescent is the “only” part; the floor outside both hoops is the neither region.

2. The key two-set formula

If you just add n(A) and n(B), the people in the overlap get counted twice. So we subtract the overlap once to fix it:

n(A ∪ B) = n(A) + n(B) − n(A ∩ B)

Rearranged, this also lets us find the overlap:

n(A ∩ B) = n(A) + n(B) − n(A ∪ B)
The four regions of a two-set diagram
  • A only = n(A) − n(A ∩ B)
  • B only = n(B) − n(A ∩ B)
  • Both = n(A ∩ B)
  • Neither = n(ξ) − n(A ∪ B)
Quick example: n(A) = 12, n(B) = 15, n(A ∩ B) = 5.
n(A ∪ B) = 12 + 15 − 5 = 22

3. Worked word problem — find how many do BOTH

Problem: In a class of 35 students, 20 offer Mathematics, 18 offer Physics, and 5 offer neither subject. How many offer BOTH?
Step 1 — at least one = total − neither = 35 − 5 = 30
Step 2 — so n(M ∪ P) = 30
Step 3 — formula: n(M ∪ P) = n(M) + n(P) − n(both)
30 = 20 + 18 − n(both) = 38 − n(both)
n(both) = 38 − 30 = 8
So 8 students offer both. Check: Maths only = 20 − 8 = 12, Physics only = 18 − 8 = 10. Regions add up: 12 + 10 + 8 + 5 neither = 35 ✓

4. Worked word problem — only one / neither

Problem: In a group of 40 people, 25 like rice, 22 like beans, and 8 like both. Find how many like (a) at least one, (b) neither, (c) only rice, (d) only beans.
(a) at least one = n(R ∪ B) = 25 + 22 − 8 = 39
(b) neither = 40 − 39 = 1
(c) rice only = 25 − 8 = 17
(d) beans only = 22 − 8 = 14
Check all regions: 17 + 14 + 8 both + 1 neither = 40 ✓
🌟 Golden rule for these problems: fill the OVERLAP first, then subtract it to get each “only” region, then use the total to get the “neither” region.

🔺 PART 4 — Three-set Venn Problems

1. The three-set formula

With three sets, three pairs overlap and there is a small centre where all three meet. We subtract each pair overlap once, but that removes the centre too many times, so we add it back:

n(A ∪ B ∪ C) = n(A) + n(B) + n(C) − n(A∩B) − n(A∩C) − n(B∩C) + n(A∩B∩C)
Quick example: n(A) = 20, n(B) = 18, n(C) = 16, n(A∩B) = 8, n(A∩C) = 6, n(B∩C) = 5, n(A∩B∩C) = 3.
Singles: 20 + 18 + 16 = 54
Pairs: 8 + 6 + 5 = 19
n(A ∪ B ∪ C) = 54 − 19 + 3 = 38

2. Filling the seven regions (the safe method)

The reliable way to solve three-set problems is to fill the diagram from the centre outward:

  • Step 1 — centre: put n(A∩B∩C) in the very middle.
  • Step 2 — pair-only regions: each pair overlap minus the centre.
    A∩B only = n(A∩B) − centre, and likewise for A∩C only and B∩C only.
  • Step 3 — single-only regions: each set total minus the three pieces already inside it.
  • Step 4 — neither: total − (sum of all seven regions).

3. Fully worked example 1

Problem: In a class of 50 students, 20 offer Maths (M), 25 offer English (E) and 18 offer Science (S). 8 offer Maths and English, 6 offer Maths and Science, 7 offer English and Science, and 3 offer all three. Find how many offer (a) at least one subject, (b) none, (c) only Maths.
Centre (all three) = 3
M∩E only = 8 − 3 = 5
M∩S only = 6 − 3 = 3
E∩S only = 7 − 3 = 4
Maths only = 20 − (5 + 3 + 3) = 20 − 11 = 9
English only = 25 − (5 + 4 + 3) = 25 − 12 = 13
Science only = 18 − (3 + 4 + 3) = 18 − 10 = 8
(a) at least one = 9 + 13 + 8 + 5 + 3 + 4 + 3 = 45
(b) none = 50 − 45 = 5
(c) only Maths = 9
Formula check: 20 + 25 + 18 − (8 + 6 + 7) + 3 = 63 − 21 + 3 = 45 ✓

4. Fully worked example 2

Problem: In a survey of 100 people about three drinks — Tea (T), Coffee (C) and Milk (K): 40 take Tea, 45 take Coffee, 30 take Milk. 15 take Tea and Coffee, 10 take Tea and Milk, 12 take Coffee and Milk, and 5 take all three. Find (a) at least one drink, (b) none, (c) only Tea.
Centre (all three) = 5
T∩C only = 15 − 5 = 10
T∩K only = 10 − 5 = 5
C∩K only = 12 − 5 = 7
Tea only = 40 − (10 + 5 + 5) = 40 − 20 = 20
Coffee only = 45 − (10 + 7 + 5) = 45 − 22 = 23
Milk only = 30 − (5 + 7 + 5) = 30 − 17 = 13
(a) at least one = 20 + 23 + 13 + 10 + 5 + 7 + 5 = 83
(b) none = 100 − 83 = 17
(c) only Tea = 20
Formula check: 40 + 45 + 30 − (15 + 10 + 12) + 5 = 115 − 37 + 5 = 83 ✓
🌟 Always finish by adding your seven regions and checking they match the number who take at least one. If the centre-outward filling and the formula agree, your answer is safe.
